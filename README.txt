FILE - CONTENT:

1) media.py:
A data structure created (Python Class) to store my favorite movies, including movie title, story line, poster URL and a YouTube link to the movie trailer.

2) entertainment_center.py: 
Multiple instances of that Python Class created to represent my favorite movies; all the instances grouped together in a list.

3) fresh_tomatoes.py:
Python module called (fresh_tomatoes.py) this module has a function called (open_movies_page) that takes in one argument, which is a list of movies and creates an HTML file which visualizes all of my favorite movies.

5) fresh_tomatoes.html:
File that opens its self automaticly ones the entertainment_center file is run.

4) README.text:
Includes details of all the steps required to succesfully run the application.





STEP TO RUN THE APPICATION:

1) Open enternteinment_center.py file (right click, edit with IDLE).
2) Run the program (run) (run module). Or just press F5. An HTML page will open automaticly to visualize some of my favorite movies.
